# <a name="policy-template-reference"></a>ポリシー テンプレート リファレンス

*適用対象:Microsoft Cloud App Security*

この記事では、Microsoft Cloud App Security に含まれるポリシー テンプレートについて説明します。

## <a name="policy-templates"></a>ポリシー テンプレート

使いやすさのため、可能な限り、既存のテンプレートに基づいてポリシー作成を開始することをお勧めします。 次の表に、Microsoft Cloud App Security に存在するポリシー テンプレートを示します。

|リスクのカテゴリ|テンプレート名|[説明]|
|-----|----|----|
|Cloud Discovery|検出されたユーザーでの異常な動作|検出されたユーザーやアプリで異常な動作が検出されたときにアラートを出します。たとえば、他のユーザーと比べてアップロードするデータ量が多い、ユーザーの履歴と比べてユーザー トランザクションの数が多い、といった場合です。|
|Cloud Discovery|検出された IP アドレスの異常なビヘイビアー|検出された IP アドレスやアプリで異常な動作が検出されたときにアラートを出します。たとえば、他の IP アドレスと比べてアップロードするデータ量が多い、IP アドレスの履歴と比べてアプリ トランザクションの数が多い、といった場合です。|
|Cloud Discovery|コラボレーション アプリのコンプライアンス チェック|SOC2 と SSAE 16 に準拠していない新しいコラボレーション アプリが検出され、そのアプリが 50 人を超えるユーザーによって使用され、1 日あたりの合計使用量が 50 MB を超えている場合にアラートを出します。|
|Cloud Discovery|クラウド ストレージ アプリのコンプライアンス チェック|SOC2、SSAE 16、ISAE 3402、PCI DSS に準拠していない新しいクラウド ストレージ アプリが検出され、そのアプリが 50 人を超えるユーザーによって使用され、1 日あたりの合計使用量が 50 MB を超えている場合にアラートを出します。|
|Cloud Discovery|CRM アプリのコンプライアンス チェック|SOC2、SSAE 16、ISAE 3402、ISO 27001、HIPAA に準拠していない新しい CRM アプリが検出され、そのアプリが 50 人を超えるユーザーによって使用され、1 日あたりの合計使用量が 50 MB を超えている場合にアラートを出します。|
|Cloud Discovery|新しいクラウド ストレージ アプリ|50 人を超えるユーザーによって使用され、1 日あたりの合計使用量が 50 MB を超える新しいクラウド ストレージ アプリが検出された場合にアラートを出します。|
|Cloud Discovery|新しいコード ホスティング アプリ|50 人を超えるユーザーによって使用され、1 日あたりの合計使用量が 50 MB を超える新しいコード ホスティング アプリが検出された場合にアラートを出します。|
|Cloud Discovery|新しいコラボレーション アプリ|50 人を超えるユーザーによって使用され、1 日あたりの合計使用量が 50 MB を超える新しいコラボレーション アプリが検出された場合にアラートを出します。|
|Cloud Discovery|新しい CRM アプリ|50 人を超えるユーザーによって使用され、1 日あたりの合計使用量が 50 MB を超える新しい CRM アプリが検出された場合にアラートを出します。|
|Cloud Discovery|ボリュームの大きい新しいアプリ|1 日あたりの合計トラフィックが 500 MB を超える新しいアプリが検出された場合にアラートを出します。|
|Cloud Discovery|アップロード ボリュームの大きい新しいアプリ|1 日あたりの合計アップロード トラフィックが 500 MB を超える新しいアプリが検出された場合にアラートを出します。|
|Cloud Discovery|新しい人事管理アプリ|新しく検出された人事管理アプリが 50 人を超えるユーザーによって使用され、1 日あたりの合計使用量が 50 MB を超えている場合にアラートを出します。|
|Cloud Discovery|新しいオンライン会議アプリ|50 人を超えるユーザーによって使用され、1 日あたりの合計使用量が 50 MB を超える新しいオンライン会議アプリが検出された場合にアラートを出します。|
|Cloud Discovery|新しい人気の高いアプリ|500 人を超えるユーザーによって使用された新しいアプリが検出された場合にアラートを出します。|
|Cloud Discovery|新しい危険なアプリ|50 人を超えるユーザーによって使用され、1 日あたりの合計使用量が 50 MB を超える、リスク スコアが 6 未満の新しいアプリが検出された場合にアラートを出します。|
|Cloud Discovery|新しい販売アプリ|50 人を超えるユーザーによって使用され、1 日あたりの合計使用量が 50 MB を超える新しい販売アプリが検出された場合にアラートを出します。|
|Cloud Discovery|新しいベンダー管理システム アプリ|50 人を超えるユーザーによって使用され、1 日あたりの合計使用量が 50 MB を超える新しいベンダー管理システム アプリが検出された場合にアラートを出します。|
|DLP|外部と共有されたソース コード|ソース コードの入ったファイルが組織外で共有された場合にアラートを出します。|
|DLP|PCI の入ったファイルがクラウドで検出されました (組み込み DLP エンジン)|承認されたクラウド アプリで、支払いカード情報 (PCI) を含むファイルが Microsoft Cloud App Security の組み込みデータ損失防止 (DLP) エンジンによって検出された場合に、アラートを出します。|
|DLP|PHI の入ったファイルがクラウドで検出されました (組み込み DLP エンジン)|承認されたクラウド アプリで、保護された医療情報 (PHI) を含むファイルが Microsoft Cloud App Security の組み込みデータ損失防止 (DLP) エンジンによって検出された場合に、アラートを出します。|
|DLP|File containing private information detected in the cloud (built-in DLP engine) (個人情報の入ったファイルがクラウドで検出されました (組み込み DLP エンジン))|承認されたクラウド アプリで、個人データを含むファイルが Microsoft Cloud App Security の組み込みデータ損失防止 (DLP) エンジンによって検出された場合に、アラートを出します。|
|脅威検出|企業 IP アドレス以外からの管理アクティビティ|管理者ユーザーが、企業 IP アドレス範囲のカテゴリに含まれない IP アドレスから管理アクティビティを実行した場合にアラートを出します。 最初に、[設定] ページに移動し、 **[IP アドレスの範囲]** を設定することで、企業 IP アドレスを構成します。|
|脅威検出|一般的な異常検出|承認されたアプリの 1 つで、異常なセッション (あり得ない移動、サインイン パターン、非アクティブなアカウントなど) が検出された場合にアラートを出します。|
|脅威検出|危険な IP アドレスからのログオン|承認されたアプリにユーザーが危険な IP アドレスからサインインした場合に、アラートを出します。 既定では、危険な IP アドレスのカテゴリには、匿名プロキシ、TOR、またはボットネットの IP アドレス タグを持つアドレスが含まれます。 IP アドレスの範囲設定ページで、このカテゴリに IP アドレスを追加することができます。|
|脅威検出|単独のユーザーによる大量ダウンロード|1 人のユーザーが 1 分以内に 50 回を超えるダウンロードを実行したときにアラートを出します。|
|脅威検出|Multiple failed user sign-in attempts to an app (1 つのアプリに対するユーザーのサインイン試行が複数回失敗)|1 人のユーザーが 1 つのアプリにサインインを試み、5 分以内に失敗回数が 10 回を超えた場合にアラートを出します。|
|脅威検出|ランサムウェアの可能性があるアクティビティ|ランサムウェアに感染している可能性があるファイルをユーザーがクラウドにアップロードした場合にアラートを出します。|
|脅威検出|カテゴリに含まれない IP アドレスからのユーザー ログオン|特定の IP 範囲カテゴリに含まれていない IP アドレスからユーザーがログオンした場合にアラートを出します。 IP アドレスを分類するには、[設定] ページに移動し、IP アドレスの範囲を選択します。|
|コントロールの共有|個人用メール アドレスで共有されたファイル|ユーザーの個人用メール アドレスでファイルが共有された場合にアラートを出します。|
|コントロールの共有|承認されていないドメインで共有されたファイル|承認されていないドメイン (競合企業のドメインなど) でファイルが共有された場合にアラートを出します。|
|コントロールの共有|共有されたデジタル証明書 (ファイル拡張子)|デジタル証明書の入ったファイルがパブリックに共有された場合にアラートを出します。 このテンプレートを使用すると、AWS ストレージを管理するのに役立ちます。|
|コントロールの共有|パブリックにアクセス可能な S3 バケット (AWS)|AWS S3 バケットがパブリックに共有された場合にアラートを出します。|
|コントロールの共有|外部共有ファイルの失効|外部共有ファイルが少なくとも 6 か月間変更されていない場合に、アラートを出します。|
