# <a name="attest-your-app"></a>アプリを証明する

Microsoft Cloud App Security を使用してアプリの構成証明を行い、クラウド アプリ カタログに収載されるアプリが最新かどうかの評価基準として使用されるコンプライアンスおよびセキュリティの詳細を確認できます。

アプリがクラウド アプリ カタログに収載済みか、それとも新規のアプリであるかにかかわらず、[自己構成証明アンケート](https://go.microsoft.com/fwlink/?linkid=2106624)を提出してください。 自己構成証明プロセスの詳細については、casfeedback@microsoft.com までお問い合わせください。

アンケートの提出を正常に完了するには、以下で説明するサービス属性に従います。

| フィールド | 情報カテゴリ | Type | 指定可能な値 | [説明] |
|------|-------|------|---------|----------|
| アプリ名 | 全般 | 文字列型 | 自由記載 | クラウド アプリ カタログに収載されるアプリケーションの名前。 |
| [説明] | 全般 | 文字列型 | 自由記載 | アプリケーションでユーザーが実行または達成できる事柄の簡単な説明。 |
| カテゴリ| 全般 | 文字列型 | クローズ リスト - アンケートで提供 | 関係するフィールドに従ったアプリの分類。 |

| データ センター| 全般 | 国番号の配列* | クローズ リスト - アンケートで提供 (複数選択) | データ センターの設置国 (複数の場所を指定可) |
| ホスティング企業 | 全般 | 文字列型 | 自由記載 | アプリをホストするサーバーを提供する会社の名前。 |
| 設立 | 全般 | 整数型 | YYYY (2019 年以降) | プロバイダーが設立された年。 |
| 株式公開状況 | 全般 | 文字列型 | Private (非公開)、Public (公開) | プロバイダーが株式公開企業か株式非公開企業かを表示します。 |
| アプリ ドメイン | 全般 | URL の配列* | 自由記載 | サービスとの対話に使用される特定のドメインの一覧。 たとえば、Microsoft Teams の場合は 'teams.microsoft.com' であり、汎用ドメインの 'microsoft.com' ではありません。 |

| ホームページ | 全般 | [URL] | 自由記載 | プロバイダーのホーム ページ URL。 |
| ディザスター リカバリー プラン | 全般 | ブール型 | True、False | バックアップ方法や復元方法を含めたディザスター リカバリー プランをこのアプリが備えているかどうか。 |

| 保存データの暗号化方法 | セキュリティ | 文字列型 | クローズ リスト - アンケートで提供 | アプリで実行される保存データ暗号化の種類。 |
| Multi-Factor Authentication | セキュリティ | ブール型 | True、False | このアプリが多要素認証ソリューションをサポートするかどうか。 |
| IP アドレス制限 | セキュリティ | ブール型 | True、False | このアプリが、アプリによる特定の IP アドレスの制限をサポートするかどうか。 |
