# <a name="cloud-app-security-documentation"></a>Cloud App Security ドキュメント

社外で共有する電子メール、ドキュメント、機密データを制御し、セキュリティの保護を容易にします。 保存先や共有するユーザーに関係なく、簡単な分類から埋め込みラベルや許可まで、Azure Information Protection で常にデータ保護を強化します。
