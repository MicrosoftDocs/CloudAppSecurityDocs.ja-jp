# <a name="troubleshooting-cloud-app-security-status"></a>Cloud App Security ステータスのトラブルシューティング

[status.cloudappsecurity.com](http://www.status.cloudappsecurity.com) にアクセスするか、ポータル内から直接、**[ヘルプ]**、**[システム ステータス]** の順にクリックすることで、Cloud App Security サービスの現在の状態を確認できます。
